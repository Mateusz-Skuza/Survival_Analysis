# Survival Analysis

This repository contains a series of projects and reports focused on the statistical analysis of survival data. The projects analyze simulated data and real medical datasets, including `lung` and `ovarian` cancer records and drug remission data.

## Report 1
This project focused on simulating survival data using the Exponentiated-Weibull distribution. Functions were developed to generate random variables and test different data censoring mechanisms, such as Type I, Type II, and random censoring, to observe their impact on survival results.

## Report 2
Kaplan-Meier and Fleming-Harrington methods were used to estimate and visualize survival probabilities over time. Statistical comparisons between patient groups were performed using various tests, including Log-rank and Peto-Peto, to identify significant differences in survival outcomes.

## Report 3
Advanced survival models were implemented to analyze how factors like age, sex, and physical health affect patient outcomes. The analysis included fitting and comparing Cox Proportional Hazards and AFT models on real-world lung cancer data to predict risks and survival probabilities.
